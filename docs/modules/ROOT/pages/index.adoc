== Installation

If you want to use this extension, you need to add the `io.quarkiverse.xxl-job:quarkus-xxl-job` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.xxl-job</groupId>
    <artifactId>quarkus-xxl-job</artifactId>
    <version>{project-version}</version>
</dependency>
----
